В папке data находятся 2 сборки. 7778_3a9748b3 большая 137 Mb, 20186_f66e4ad3 маленькая 5Mb

Рендеринг происходит с помощью библиотеки trimesh. Ее стоит скачать.

> pip install trimesh

Запускать скрипт coloring.py нужно с аргументом имени сборки. Пример:

> python coloring.py 7778_3a9748b3

> python coloring.py 20186_f66e4ad3

После запуска появится окно с рендерингом, если оно черное или некрутиться, мне помогало подождать и/или двойное нажатие по левой клавиши мышки.

Скрипт preprocess_normal_depth.py рендерит нормали и глубину меша сборки, но он только для визуализации, так как значения глуюины и нормали я брала в мировых координатах, а не относительно точки обзора. Его нужно запускать с аргументом имени сборки и режимом рендеринга normal или depth. Пример:
> python preprocess_normal_depth.py 7778_3a9748b3 normal
